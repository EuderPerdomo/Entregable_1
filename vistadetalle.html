Preguntas Frecuentes

  • ¿Puedo Trabajarlo sin Baterias?

    No, no es posible iniciar el equipo sin baterias

  • ¿Puedo unir el equipo a otros para aumentar la potencia o formar circuitos trifasicos?

    Es posible realizar la unión de varios de estos equipos en paralelo de tal manera que podemos incrementar la potencia de salida o tambien tener la posibilidad de formar un circuito trifasico. (Si este es tu caso recuerda indicarselo a tu asesor de ventas ya que para esta funcionalidad se requieren algunos componentes adicionales.)

  • ¿Puedo inyectar a la red con este dispositivo?

    No, este dispositivo no admite la funcionalidad de inyeccion, solamente permite la entrada de red como una utilidad

  • ¿El equipo realiza el bypass automatico en caso de cortes de energía?

    Si, el dispositivo tiene la capacidad de realizar bypass automatico sin que se afecte el funcionamiento de las cargas conectadas.
